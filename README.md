# 🤖 Basic Chatbot

A simple and interactive **Python-based Chatbot** that can answer programming-related questions, tell jokes, display motivational quotes, perform calculations, open websites, and save chat history.

This project demonstrates Python fundamentals such as file handling, modular programming, dictionaries, lists, functions, loops, and exception handling.

---

## 📌 Features

- 👋 Greets the user
- 💬 Interactive conversation
- 📖 Answers programming-related questions
- 😂 Tells random programming jokes
- 💡 Displays motivational quotes
- 🧮 Performs mathematical calculations
- 🌐 Opens popular websites
- 📅 Shows current date
- 🕒 Shows current time
- 📝 Saves chat history
- 🗑️ Clears chat history
- 🔍 Suggests similar commands for incorrect input
- 🚪 Exit command

## ▶️ How to Run

```bash

```

### 2. Open the project folder

```bash
cd Basic-Chatbot
```

### 3. Run the chatbot

```bash
python chatbot.py
```
